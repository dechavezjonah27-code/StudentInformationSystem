# Student Information System

## Description
This is a simple Student Information System using HTML, CSS, JavaScript, PHP, and MySQL.
It allows users to add, view, edit, and delete students.

## Features
- Add new student (first name, last name, email)
- View all students in a table
- Edit student information
- Delete student
- Dynamic interaction using JavaScript fetch() (works on server)
